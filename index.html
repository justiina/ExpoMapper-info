Overview

ExpoMapper is a research-oriented tool designed to support the organization, mapping, and exploration of exposome-related data. It aims to facilitate reproducible and transparent analysis workflows in environmental and health research.

Key Features

  • Structured representation of exposome data
  • Support for harmonized variable mapping
  • Designed for research and academic use

Getting Started

Documentation and usage examples are available on the Documentation page. The source code is hosted on GitHub.
